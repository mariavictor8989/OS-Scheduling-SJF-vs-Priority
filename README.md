# OS Scheduling Algorithms Simulator

A Java Swing application that simulates and compares two CPU scheduling algorithms:
* **SJF (Shortest Job First)**
* **Priority Scheduling**

## Features
* **Modular Design:** Separate panels for input, Gantt charts, and results.
* **Responsive UI:** Dark green themed interface with dynamic table updates.
* **Gantt Charts:** Visual representation of process execution.
* **Performance Metrics:** Calculates Waiting Time, Turnaround Time, and Response Time.

## How to Run
1. Clone the repository.
2. Open the project in NetBeans or any Java IDE.
3. Run the `SJF_vs_Priority.java` file.
