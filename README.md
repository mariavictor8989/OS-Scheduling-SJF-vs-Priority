# OS Scheduling Algorithms Simulator
**A Comprehensive Java-based Visualization Tool for SJF and Priority Scheduling.**

###  Key Features
* **Interactive UI:** Professional dark-green theme with dynamic input tables.
* **Gantt Charts:** Real-time, color-coded visual execution timeline.
* **Dual Modes:** Supports both Non-Preemptive and Preemptive (SRTF/Priority).
* **Data Integrity:** Robust validation to catch negative times and invalid characters.

###  Performance Metrics
The simulator provides a deep dive into the following metrics:
* **AWT:** Average Waiting Time.
* **ATAT:** Average Turnaround Time.
* **RT:** Response Time.

### How to Run
1. **Clone the Repository:**
   `git clone https://github.com/mariavictor8989/OS-Scheduling-SJF-vs-Priority.git`
2. **Open Project:** Open the project in NetBeans or any Java IDE.
3. **Run:** Execute the `SJF_vs_Priority.java` file.
